# 📱 Hungarian Hardstyle App

The official mobile application of **Hungarian Hardstyle**.

The app aims to become the central platform of the Hungarian harder styles scene by bringing together news, events, artists, organizers, releases and the community in a single application.

# 🎯 Vision

Hungarian Hardstyle aims to become the central platform of the Hungarian harder styles community by combining:

- 📰 News
- 📅 Events
- 🎧 DJs
- 🏢 Organizers
- 🎵 Hardstyle Revolution Releases
- 📻 Online Radio
- ❤️ Community
- 🛒 Digital Music Distribution

into a single cross-platform mobile application for **Android** and **iOS**.
